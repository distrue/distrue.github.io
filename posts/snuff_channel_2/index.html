스누푸파 카카오채널 (2) AWS Lambda로 응답 API 구축

지난 번 post 에서 크롤링한 데이터를 mongoDB에 저장해둔 후, 이를 스누푸파 카카오채널에서 응답할 수 있도록 AWS Lambda를 이용해 채널을 개발하였다.

카카오채널의 스킬 응답을 구성하기 위해서는 openbuilder-node 라이브러리를 활용하였다.

  • 이 라이브러리는 직접 제작한 라이브러리로 지금은 v.0.3 인데, 일반적으로 활용되는 카카오채널의 스킬 응답 / 요청 / 검증 API를 객체화해 사용할 수 있도록 구축해 두었다. 사용해보고 마음에 들면 스타 🌟 한번…ㅎㅎ

3. API gateway와 연결

Lambda가 호출될 수 있도록 API endpoint를 추출하자.

다음과 같이 amazon console에서 API Gateway로 접근한 후 HTTP API를 생성한다.

  1. 통합을 lambda로 연결, 기존에 생성한 Lambda function과 연결해준다.
  2. 경로 구성에서 POST /snuff_keyword_rcmd 와 실제 lambda function을 연결해준다.
  3. $default 스테이지로 두고 API Gateway를 생성한다.

이제 Lambda로 가보면 API endpoint가 연결된 것을 확인할 수 있다.

Lambda 에서 모니터링 > 로그 > CloudWatch에서 로그보기 로 접근하면 cloudwatch 에서 lambda의 실행 결과를 확인할 수 있다.

  • 디버깅을 해보면서 잘 동작하지 않으면 빠진 부분이 있지는 않은지 확인해보자
